NASA picture a day

A project using react and API calls to the NASA API to display the daily picture and description. You can use the calendar to select a different day and see the page for that day. I have the image displayed again behind the main image as well.

Citrics back-end

Takes user preferences and generates a list of relevant cities based on criteria such as weather, crime rate and location. Using Spring boot collaboratively created a RDBMS to store user data pulled in from the react front-end. Collaborated remotely with 5 other web Developers and 5 data science engineers to build a fully functional application for our stake holder.
HTML/CSS | JavaScript | React | Java Spring | Fast API
